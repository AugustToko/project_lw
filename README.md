# project_lw


## 动态壁纸管理器


目前支持
> HTML  
> Video  
> View  
> Image  


Project LW 用作与 Android 平台上壁纸管理，支持 HTML 视频 自定义视图 图片 五种类型做壁纸。同时具备创意工坊功能，玩家可以上传自定义的壁纸到服务器用作分享用户制作的壁纸。
